# BookVerse

**BookVerse** is an online book reading platform where users can browse books, track their reading progress, and leave reviews. All books are read **online** — no downloads — allowing seamless progress tracking and a smooth reading experience.

---

## Features

- **User Management**
  - User registration, login/logout
  - Role-based access (Admin / Normal user)
  - Password reset

- **Books**
  - Browse all books
  - Search by title, author, or keyword
  - Filter by category/genre
  - View book details: title, author, cover image, description, category, average rating
  - Read books online with an embedded viewer
  - Track reading progress (current page and percentage)
  - “Continue Reading” section for in-progress books

- **Reviews & Ratings**
  - Users can rate books (1–5 stars)
  - Users can leave text reviews
  - One review per user per book
  - Average rating displayed for each book

- **Admin Features**
  - Upload, edit, delete books
  - Manage categories
  - Moderate reviews
  - Manage users

- **UI / UX**
  - Responsive design (desktop, tablet, mobile)
  - Intuitive navigation

---
